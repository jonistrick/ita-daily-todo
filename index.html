Ein Daily-Tool für den echten ITA-Arbeitsalltag.

Das ITA Daily To-do Tool bündelt Jira-Tickets aus mehreren Zeitfeldern in einer gemeinsamen, chronologischen Tagesansicht. Aufträge mit Einsatzzeit, ReserWUtion mit Start Time und Events mit Assembly Time Start beziehungsweise Dismantling Time landen so in einer einzigen Liste statt in mehreren getrennten Suchen.

Enthaltene Kernfunktionen

  • Today-Preset als Standardansicht
  • Preset -14 bis +3 für einen breiteren Zeitraum
  • Erweiterter Modus mit individuellen Start- und Endtagen
  • Lokale Done-Häkchen pro Tag mit Fokus auf die erste offene Aufgabe
  • Benachrichtigung bei neuen Heute-Tickets per Badge und Browser-Notification

Installation in Edge

Die Erweiterung wird lokal als entpackte Erweiterung geladen. So kann sie intern schnell verteilt und direkt mit der bestehenden Jira-Session genutzt werden.

Schritt für Schritt

  1. ZIP-Datei herunterladen und in einen normalen Ordner entpacken.
  2. In Edge edge://extensions öffnen.
  3. Den Entwicklermodus aktivieren.
  4. Auf Entpackte Erweiterung laden klicken.
  5. Den entpackten Ordner des ITA Daily To-do Tools auswählen.
  6. Die Erweiterung an die Symbolleiste anheften und das Popup öffnen.

Erster Einsatz

  • Die Erweiterung verwendet die bestehende Jira-Browser-Session.
  • Die erste Ansicht ist standardmäßig Heute.
  • Über Einstellungen lassen sich Basis-URL, JQL und Feld-IDs anpassen.
  • Neue Einträge in der Heute-Liste werden im Hintergrund erkannt und gemeldet.

Funktionen

Das Tool ist für die tägliche Arbeit mit unterschiedlichen Ticketarten gebaut und verbindet mehrere Jira-Zeitlogiken in einer einzigen nutzbaren Übersicht.

Gemeinsame Zeitlogik

Das Tool sortiert nicht nur nach einem einzelnen Feld, sondern führt Aufträge, ReserWUtion und Events mit ihren jeweils passenden Zeitstempeln in einer chronologischen Liste zusammen.

Aufträgeüber Einsatzzeit
ReserWUtionüber Start Time
Eventsüber Aufbau und Abbau

Lokaler Arbeitsmodus

Done-Häkchen bleiben lokal im Browser gespeichert und sind pro Tag getrennt. So kann die Liste als persönliche Arbeitsansicht verwendet werden, ohne den Jira-Datensatz selbst zu verändern.

Benachrichtigung bei neuen Tickets

Das Tool prüft die Heute-Ansicht regelmäßig im Hintergrund. Wenn ein neues Ticket in der Liste auftaucht, erscheint ein Badge an der Erweiterung und zusätzlich eine Browser-Benachrichtigung, damit kein Eintrag übersehen wird.

Auf einen Blick

Die Erweiterung verbindet eine klare Tagesansicht mit lokalen Arbeitsfunktionen und einer unaufdringlichen Erinnerung an neue Tickets.
