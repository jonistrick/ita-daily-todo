Eine Daily-To-do-Liste, die wirklich zum ITA-Workflow passt.

Das ITA Daily To-do Tool ist eine lokale Browser-Erweiterung für Jira Data Center. Sie zieht Aufträge über Einsatzzeit, ReserWUtion über Start Time und Events über Assembly Time Start sowie Dismantling Time in eine gemeinsame, chronologische Tagesansicht.

Was das Tool im Alltag besser macht

Statt mehrere Jira-Suchen offen zu haben, zeigt das Tool die relevanten Tagesaufgaben in einer einzigen Reihenfolge. Gerade bei wechselnden Zeitfeldern und unterschiedlichen Tickettypen spart das täglich manuellen Sortieraufwand.

3

Ticketarten werden gemeinsam abgebildet: Aufträge, ReserWUtion und Events.

2

Presets stehen direkt oben bereit: Heute und -14 bis +3, plus ein erweiterbarer Bereich.

1

Daily-Ansicht statt Filterwechsel: ein Blick genügt, um die Reihenfolge des Tages zu sehen.

Funktionslogik

Die Seite beschreibt jetzt die reale Logik des Tools und zeigt auch optisch ein Popup, das eng an dein echtes Interface angelehnt ist. Damit kannst du die GitHub Page bereits veröffentlichen, ohne dass es nur wie ein generischer Mockup-Entwurf wirkt.

Gemeinsame Sortierung

Die Extension nimmt pro Ticketart das jeweils richtige Zeitfeld und bringt daraus eine Daily-Liste hervor, die für den tatsächlichen ITA-Arbeitsablauf besser passt als eine einzelne Standard-Jira-Sortierung.

Lokale Daily-Helfer

Einträge können lokal als erledigt markiert werden, ohne Jira selbst zu verändern. Dadurch bleibt die Liste operativ nutzbar, auch wenn in Jira keine direkte Anpassung möglich ist.

Benachrichtigungen

Im Hintergrund kann das Tool neue Einträge für die Heute-Ansicht erkennen und Badge plus Notification auslösen, auch wenn das Popup nicht offen ist.
